# Retail Sales Analytics and Business Insights using MySQL

## 📌 Project Overview

This project analyzes retail sales transaction data using MySQL to uncover valuable business insights. The analysis focuses on customer purchasing behavior, sales performance, revenue trends, and category-wise business growth.

The project demonstrates SQL skills commonly used in Data Analytics and Business Intelligence roles, including data cleaning, aggregations, Common Table Expressions (CTEs), window functions, and business reporting.

---

## 🎯 Objectives

* Clean and validate retail sales data.
* Analyze category-wise sales performance.
* Identify top customers and purchasing patterns.
* Determine monthly and yearly sales trends.
* Generate actionable business insights using SQL.
* Practice advanced SQL concepts on a real-world dataset.

## 📂 Dataset Schema

| Column Name     | Description             |
| --------------- | ----------------------- |
| transactions_id | Unique transaction ID   |
| sale_date       | Date of transaction     |
| sale_time       | Time of transaction     |
| customer_id     | Customer identifier     |
| gender          | Customer gender         |
| age             | Customer age            |
| category        | Product category        |
| quantity        | Quantity purchased      |
| price_per_unit  | Product price per unit  |
| cogs            | Cost of goods sold      |
| total_sale      | Total transaction value |
